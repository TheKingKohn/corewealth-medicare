This folder is for storing images used on your website.

### Recommended Images:

1. **Agent Photo** (`agent-photo.jpg` or `.png`)
   - Professional headshot
   - Recommended size: 400x400px to 800x800px
   - Square format works best

2. **Logo** (if you have one) (`logo.png`)
   - Transparent background recommended
   - Recommended size: 200-300px wide

3. **Optional Images:**
   - Office photo
   - Team photos
   - Additional graphics

### How to Use:

Once you add images to this folder, update the HTML file:

```html
<!-- Replace the placeholder in index.html with: -->
<img src="images/agent-photo.jpg" alt="Agent Name" style="border-radius: 15px; width: 100%; max-width: 400px;">
```

### Image Optimization Tips:

- Keep file sizes under 500KB for fast loading
- Use JPEG for photos, PNG for logos with transparency
- Consider using online tools like TinyPNG or Squoosh to compress images
